#  ArtStore - Web store for local individual artist/brand.
- Website solution for local **artists/craftsmen/brands** selling their products.

- 1. Home page
- 2. Product Detail View
- 3. Cart 
- 4. Direct Order(Buy Now)

- 6. Payment features
